Behind this Easter Bunny Tracker, there are many products, tool, and people that have helped in the development of this tracker, and I'd love to acknowledge everything that makes this tracker possible.

Tools/Libraries:

  • The React framework (https://reactjs.org/)
  • Material Design Bootstrap v4 and v5 (https://mdbootstrap.com/)
  • Axios (https://axios-http.com/)
  • Bowser v1 & v2 (https://github.com/lancedikson/bowser)
  • MUI Icons (https://mui.com/material-ui/material-icons/)
  • Flag Icons (https://flagicons.lipis.dev/)
  • Luxon (https://moment.github.io/luxon/) & Moment.JS (https://momentjs.com/)
  • Weather Icons (https://erikflowers.github.io/weather-icons/)
  • Polyfill.io (https://polyfill.io/v3/)
Services:
  • CDNJS
  • Cloudflare
  • GeoNames
  • Google Maps API
  • Google Timezone API
  • Apple WeatherKit API
  • Mapbox Reverse Geocoding API
  • Wikipedia for stop population, elevation, and descriptions.
The basket icon used in the tracker is provided by Roundicons.

The animation used when the Easter Bunny is at a stop was created by COLSTER.

The track.easterbunny.cc Version 4 team, which overhauled the tracker for 2020 out of a prototype and into a fully functional tracker & website and set the path forward for this tracker:
  • Owen (Lead Developer & QA Engineer) - Ran the entire TEBCC v4 team, and led tracker development.
  • Jamie (Routing Engineer) - Worked extensively with the native Wikipedia integrations, and cleaning up pronunciations as well.
  • Carl (User Acceptance Testing) - Helped to ensure that the tracker was accessible and simple to use.
  • Jack (QA Engineer) - Helped us expand our quality assurance to 11,000+ runs to ensure the tracker is virtually bug-free.
  • Sonic (Social Media Manager) - Helped to run the @bunny_tracking twitter, and advertise the tracker.
And lastly, the entire tracking community who isn't afraid to give us feedback to help make the tracker the best it can be.
